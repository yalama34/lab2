## Краткая справка по использованию
<pre>
ls <источник> - отображение элементов в выбранном пути. Ключи: -l - подробное отображение информации об элементах
cd <источник> - переход по указанной директории
cat <источник> - вывод содержимого файла
cp <источник> <назначение> - копирование файла из источника в назначение. Ключи: -r - рекурсивное копирование каталога
mv <источник> <назначение> - перемещение файла из источника в назначение
rm <источник> - удаление указанного элемента. Ключи: -r - рекурсивное удаление директории
zip <каталог> <название.zip> - создание ZIP архива из выбранного каталога
unzip <название.zip> - распаковка ZIP архива в текущую директорию
tar <каталог> <название.tar> - создание TAR.GZ архива из выбранного каталога
untar <название.tar> - распаковка TAR.GZ архива в текущую директорию
grep <паттерн> <источник> - поиск строк, соответствующих заданному паттерну в файлах. Ключи: -i - поиск без учета регистра, -r - рекурсивный поиск
</pre>
